# admin-boilerplate

Open-source admin-area boilerplate for SaaS-style projects. Cloudflare Worker (SPA + API), Tauri desktop wrapper, magic-link auth, Swiss/Apple design.

See [`docs/superpowers/specs/2026-05-13-admin-boilerplate-design.md`](docs/superpowers/specs/2026-05-13-admin-boilerplate-design.md) for the full design.

> 🚧 This boilerplate is under active construction. The README quickstart is added in the foundation plan.
